# RPG Token Borders

Free SVG border frames for tabletop RPG character tokens. Use them with any virtual tabletop — Roll20, Foundry VTT, Fantasy Grounds, Owlbear Rodeo, and more.

## Preview

| Fantasy | Minimal | Sci-Fi |
|---------|---------|--------|
| Classic gold rings, ornate frames | Clean circles, thin strokes | Neon rings, tech frames |

## What's Included

- **Fantasy** — 8 borders (ornate, royal, arcane, etc.)
- **Minimal** — 4 borders (clean, thin, modern)
- **Sci-Fi** — 3 borders (neon, tech, holographic)

All borders are:
- SVG format with `viewBox="0 0 512 512"`
- Transparent centers (ready for portrait overlay)
- Optimized for VTT grid sizes (280px–400px+)

## Usage

1. Download any SVG from the `borders/` folder
2. Overlay your character portrait in any image editor
3. Export as PNG/WebP at your VTT's recommended size

Or use them instantly with **[RPG Token Maker](https://rpgtokenmaker.com)** — a free browser-based tool that lets you batch-create VTT-ready tokens with these borders in seconds.
